<task type="auto">
  <name>Task 2: Implement classify() to pass all tests</name>
  <files>src/classifier.js</files>
  <action>
    Replace the stub body of `classify()` with full implementation.
    Keep the module.exports line unchanged. Keep the function signature unchanged.

    **Step 1 — Eligibility check:**
    ```js
    const FIX_KEYWORDS = [
      'fix', 'broken', 'error', 'bug', 'not working', 'failing',
      'crash', 'issue', 'wrong', 'help', 'why is', "why isn't",
      "why doesn't", "can't get", "doesn't work"
    ];
    const IMPERATIVE_VERBS = ['update', 'refactor', 'rewrite', 'debug', 'check'];
    const CODE_NOUNS = ['function', 'component', 'class', 'module', 'file', 'code', 'script', 'test'];
    ```

    Eligibility: `lower.includes(keyword)` for any FIX_KEYWORDS entry,
    OR prompt starts with IMPERATIVE_VERB followed by a CODE_NOUN (check with split/indexOf).
    If not eligible: return `{ shouldInject: false, tier: 1, score: 0, signals: [] }`.

    **Step 2 — Score each signal (add to score, push signal key to signals array):**

    | Signal key | Detection | Points |
    |---|---|---|
    | `still_again` | `/\b(still|again)\b/i.test(text)` | +2 |
    | `rhetorical_why` | `/\bwhy\b/i.test(text)` | +1 |
    | `excessive_punctuation` | `/\?\?/.test(text) || /!!/.test(text)` | +2 |
    | `all_caps` | `/\b[A-Z]{3,}\b/.test(text)` | +2 |
    | `deadline_words` | `/\b(demo|deploy|prod|launch|presentation|client|boss|interview)\b/i.test(text)` | +3 |
    | `existential_words` | `/\b(please|begging|desperate|dying|killing me|hate this)\b/i.test(text)` | +2 |
    | `time_of_day` | `const h = (ctx.timestamp || new Date()).getHours(); h >= 23 || h <= 3` | +2 |
    | `file_repeat` | `Object.values(ctx.fileMentionCounts || {}).some(c => c >= 3)` | +2 |
    | `quoted_error` | `text.includes('"')` | +1 |
    | `long_prompt` | `text.length > 300` | +1 |

    **Step 3 — Score to tier:**
    ```js
    function scoreToTier(score) {
      if (score >= 9) return 5;
      if (score >= 6) return 4;
      if (score >= 4) return 3;
      if (score >= 2) return 2;
      return 1;
    }
    ```

    **Return:** `{ shouldInject: true, tier, score, signals }`

    Avoid: mutating input, adding any console.log, using external modules.
    Avoid: changing the function signature or module.exports shape.
  </action>
