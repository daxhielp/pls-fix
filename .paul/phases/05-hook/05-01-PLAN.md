    Replace the TODO comment inside `process.stdin.on('end', () => { ... })` with the full implementation. Keep the shebang, 'use strict', all existing imports, and the DRY_RUN constant unchanged.

    **Full handler body:**

    ```js
    process.stdin.on('end', () => {
      let input;
      try {
        input = JSON.parse(raw);
      } catch {
        process.exit(0);
      }

      const promptText = input.prompt || '';
      const cwd = input.cwd || '';

      const session = readSession();
      const context = {
        fileMentionCounts: session.filePleaCounts,
        timestamp: new Date(),
      };

      let result;
      try {
        result = classify(promptText, context);
      } catch {
        process.exit(0);
      }

      if (result.shouldInject) {
        const phrase = tiers[result.tier];
        if (DRY_RUN) {
          process.stderr.write(
            `[pls-fix dry-run] Would have appended (Tier ${result.tier}):\n  "${phrase}"\n` +
            `⚠️  Prompt sent without blessing. Mistake prevention not guaranteed.\n`
          );
        } else {
          process.stdout.write(phrase + '\n');
        }
      }

      updateSession(result, promptText, cwd);
      process.exit(0);
    });
    ```

    Key behaviors:
    - JSON.parse failure → exit 0 silently (Claude must not be blocked by hook errors)
    - classify() failure → exit 0 silently (same reason)
    - `input.prompt` may be missing on edge cases → default to empty string
    - `input.cwd` may be absent → default to empty string (session.js handles empty key)
    - DRY_RUN: write to stderr, NOT stdout — stdout must stay clean for injection protocol
    - DRY_RUN: updateSession still called — tracking continues even when injection skipped
    - Non-eligible prompts: updateSession still called (increments promptsPassedThrough)
    - No console.log anywhere — stdout is the injection channel, must stay clean

    Avoid: writing anything to stdout when shouldInject is false. Avoid: process.exit(1) on recoverable errors — all exits must be 0 so Claude continues normally. Avoid: importing or using any module not already in the file.
