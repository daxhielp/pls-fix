<task type="auto">
  <name>Task 1: Implement readSession, writeSession, resetSession, and updateSession</name>
  <files>src/session.js</files>
  <action>
    Replace all TODO stubs with full implementations. Keep all existing constants (SESSION_FILE, STREAK_FILE, FRESH_SESSION) and module.exports unchanged.

    **readSession():**
    ```js
    function readSession() {
      try {
        return JSON.parse(fs.readFileSync(SESSION_FILE, 'utf8'));
      } catch {
        return FRESH_SESSION();
      }
    }
    ```

    **writeSession(sessionObj):**
    ```js
    function writeSession(sessionObj) {
      fs.writeFileSync(SESSION_FILE, JSON.stringify(sessionObj, null, 2));
    }
    ```

    **resetSession():**
    ```js
    function resetSession() {
      writeSession(FRESH_SESSION());
    }
    ```

    **updateSession(classifyResult, promptText, cwd):**
    1. `const session = readSession()`
    2. If `classifyResult.shouldInject` is false:
       - Increment `session.promptsPassedThrough`
       - Write and return
    3. If `classifyResult.shouldInject` is true:
       - Increment `session.promptsBlessed`
       - Set `session.mistakesPrevented = session.promptsBlessed`
       - Increment `session.tierBreakdown[String(classifyResult.tier)]`
       - Update `session.filePleaCounts[cwd]`: increment by 1 (or set to 1 if absent)
       - Build candidate = `{ text: promptText, tier: classifyResult.tier, score: classifyResult.score, timestamp: new Date().toISOString(), file: cwd }`
       - If `session.mostDesperatePrompt === null` OR `candidate.score > session.mostDesperatePrompt.score`:
         - Set `session.mostDesperatePrompt = candidate`
    4. `writeSession(session)`

    Avoid: throwing on missing file (readSession handles that). Avoid: JSON.parse without try/catch in readSession (already shown above). Avoid: mutating FRESH_SESSION object directly (it's a factory function — always call FRESH_SESSION() to get a fresh copy).
  </action>
  <verify>
    Run:
    ```
    node -e "const s = require('./src/session'); s.resetSession(); console.log(s.readSession().promptsBlessed)"
    ```
    Expected output: `0`

    Run:
    ```
    node -e "const s = require('./src/session'); s.resetSession(); s.updateSession({shouldInject:true,tier:2,score:3,signals:[]}, 'fix my bug', '/tmp'); const sess = s.readSession(); console.log(sess.promptsBlessed, sess.tierBreakdown['2'], sess.mostDesperatePrompt.tier)"
    ```
    Expected output: `1 1 2`
  </verify>

  <files>src/session.js</files>
  <action>
    Add two new exported functions below the existing four. Do not change any existing functions.

    **readStreak():**
    ```js
    function readStreak() {
      try {
        return JSON.parse(fs.readFileSync(STREAK_FILE, 'utf8'));
      } catch {
        return { currentStreak: 0, personalBest: 0, lastSessionDate: null };
      }
    ```

    **writeStreak(streakObj):**
    ```js
    function writeStreak(streakObj) {
      const dir = path.dirname(STREAK_FILE);
      if (!fs.existsSync(dir)) fs.mkdirSync(dir, { recursive: true });
      fs.writeFileSync(STREAK_FILE, JSON.stringify(streakObj, null, 2));
    }
    ```

    Update module.exports to include both new functions:
    ```js
    module.exports = { readSession, writeSession, resetSession, updateSession, readStreak, writeStreak };
    ```
