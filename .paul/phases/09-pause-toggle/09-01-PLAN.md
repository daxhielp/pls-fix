# Phase 9 Plan 01: Pause Toggle via Slash Commands

## Goal

Add runtime pause/resume of pls-fix injection via Claude Code slash commands `/pls-fix-pause` and `/pls-fix-resume`, replacing the current workflow of reinstalling with `--dry-run` to disable injection.

## Mechanism

- **Flag file**: `~/.pls-fix/paused` ‚Äî empty file; presence means injection is paused
- **hook.js**: checks for flag file before writing to stdout; skips injection if paused; still calls `updateSession()` so session stats continue accumulating
- **cli.js**: `pause` subcommand creates flag file; `resume` subcommand removes it; `install` writes slash command `.md` files into `~/.claude/commands/`; `uninstall` removes them
- **Slash command files**: `.md` files that, when invoked as `/pls-fix-pause` or `/pls-fix-resume`, direct Claude to run the cli command with the absolute path baked in at install time

## Acceptance Criteria

- AC-1: `hook.js` skips stdout injection (but still calls `updateSession()`) when `~/.pls-fix/paused` exists
- AC-2: `pls-fix pause` creates `~/.pls-fix/paused`; `pls-fix resume` removes it
- AC-3: `pls-fix install` writes `~/.claude/commands/pls-fix-pause.md` and `~/.claude/commands/pls-fix-resume.md` with absolute path to cli.js baked in
- AC-4: `pls-fix uninstall` removes both slash command files
- AC-5: Usage line updated to include pause/resume

### Task 2 ‚Äî Update cli.js: pause/resume subcommands + slash command management

**Action:** Three changes to `cli.js`:

**2a ‚Äî Add PAUSED_FILE and COMMANDS_DIR constants** after `SESSION_FILE`:
```javascript
const PLS_FIX_DIR  = path.join(os.homedir(), '.pls-fix');
const PAUSED_FILE  = path.join(PLS_FIX_DIR, 'paused');
const COMMANDS_DIR = process.env.CLAUDE_CONFIG_DIR
  ? path.join(process.env.CLAUDE_CONFIG_DIR, 'commands')
  : path.join(os.homedir(), '.claude', 'commands');
```

**2b ‚Äî Add `pause` and `resume` cases** to the switch block (before `default`):
```javascript
case 'pause': {
  fs.mkdirSync(PLS_FIX_DIR, { recursive: true });
  fs.writeFileSync(PAUSED_FILE, '');
  console.log('[pls-fix] Injection paused. Run: pls-fix resume (or /pls-fix-resume) to re-enable.');
  break;
}
case 'resume': {
  if (fs.existsSync(PAUSED_FILE)) fs.unlinkSync(PAUSED_FILE);
  console.log('[pls-fix] Injection resumed. Blessings will flow again.');
  break;
}
```

**2c ‚Äî Slash command file management in `install` and `uninstall`:**

In the `install` case, after `writeConfig(config)` and `~/.pls-fix/` dir creation, add:
```javascript
const cliPath = path.resolve(__dirname, 'cli.js');
fs.mkdirSync(COMMANDS_DIR, { recursive: true });
fs.writeFileSync(
  path.join(COMMANDS_DIR, 'pls-fix-pause.md'),
  `Run this command to pause pls-fix injection:\n\n\`\`\`bash\n${nodeExec} ${cliPath} pause\n\`\`\`\n\nAfter it completes, reply only with: "pls-fix injection paused ‚è∏"\n`
);
fs.writeFileSync(
  path.join(COMMANDS_DIR, 'pls-fix-resume.md'),
  `Run this command to resume pls-fix injection:\n\n\`\`\`bash\n${nodeExec} ${cliPath} resume\n\`\`\`\n\nAfter it completes, reply only with: "pls-fix injection resumed ‚úì"\n`
);
console.log('[pls-fix] Slash commands registered: /pls-fix-pause, /pls-fix-resume');
```

In the `uninstall` case, after `writeConfig(config)` and before `SESSION_FILE` unlink, add:
```javascript
[
  path.join(COMMANDS_DIR, 'pls-fix-pause.md'),
  path.join(COMMANDS_DIR, 'pls-fix-resume.md'),
].forEach(f => { try { fs.unlinkSync(f); } catch {} });
```

Update the `default` usage line:
```javascript
console.log('Usage: pls-fix <install|uninstall|report|pause|resume> [--dry-run]');
```

**Verify:** Run `node cli.js pause` ‚Äî `~/.pls-fix/paused` created. Run `node cli.js resume` ‚Äî file removed. Reinstall under `CLAUDE_CONFIG_DIR` isolation and confirm both `.md` files appear in commands dir with correct absolute paths. Run uninstall and confirm they're removed.

**Done:** pause/resume subcommands work; slash command files written on install; removed on uninstall; usage updated.
