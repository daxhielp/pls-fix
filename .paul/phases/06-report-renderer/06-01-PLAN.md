    **1. Add shebang and update the top of the file:**
    ```js

    ```
    Remove the old `require('./session')` line that only imports `readSession`.

    **2. Add `updateStreak()` internal function (not exported) before `printReport()`:**

    Logic:
    - Read streak with `readStreak()`
    - Get today as `new Date().toISOString().slice(0, 10)` (YYYY-MM-DD string)
    - Compare `streak.lastSessionDate` to today:
      - If `streak.lastSessionDate === today`: do nothing — same-day second run, no increment
      - Else if `streak.lastSessionDate` is yesterday (diff = 1 day): increment `streak.currentStreak`
      - Else (null, or >1 day gap): reset `streak.currentStreak = 1`
    - If `streak.currentStreak > streak.personalBest`: update `streak.personalBest`
    - Set `streak.lastSessionDate = today`
    - Write streak with `writeStreak(streak)`
    - Return the updated streak object

    To check "yesterday": compare the date strings by converting to Date objects:
    ```js
    const last = new Date(streak.lastSessionDate);
    const now = new Date(today);
    const diffDays = Math.round((now - last) / 86400000);
    ```
    If diffDays === 1 → increment. If diffDays === 0 → no-op. If diffDays > 1 or NaN → reset to 1.

    Avoid: throwing if streak file is corrupt — readStreak() already returns defaults on error.
  </action>

       ```
       top
       row('           pls-fix  session report        ')
       mid
       row(`  Prompts blessed          ${session.promptsBlessed}`)
       row(`  Mistakes prevented       ${session.promptsBlessed}  (${pct}%)`)
       ```

       - Close with `mid` if section rendered
    6. **Most desperate prompt section:**
       - If `session.mostDesperatePrompt` exists:
         - Format timestamp: `new Date(mdp.timestamp).toLocaleTimeString('en-US', {hour:'numeric', minute:'2-digit'})` → e.g. "5:11 PM"

         - Add each wrapped line as `row('  ' + chunk)`
       - `bottom`
    7. Print all lines with `console.log(lines.join('\n'))`
    8. **Streak footer** (after the box, no border):
       ```js
       const pb = streak.currentStreak >= streak.personalBest && streak.personalBest > 0 ? '  ✦ personal best' : '';
       const dayWord = streak.currentStreak === 1 ? 'day' : 'days';
       console.log(`\n  Current no-mistake streak: ${streak.currentStreak} ${dayWord}${pb}`);
       ```

    **Word-wrap helper** for prompt text:
    ```js
    function wrap(text, width) {
      const words = text.split(' ');
      const lines = [];
      let cur = '';
      for (const w of words) {
        if ((cur + (cur ? ' ' : '') + w).length > width) {
          if (cur) lines.push(cur);
          cur = w;
        } else {
          cur = cur ? cur + ' ' + w : w;
        }
      }
      if (cur) lines.push(cur);
      return lines;
    }
    ```
